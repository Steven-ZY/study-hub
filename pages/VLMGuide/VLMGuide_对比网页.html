一、VLA 在部署中遇到的 OOD 类型

1.1 真机实际观察（松灵双臂 π0.5，一手数据）

#失败现象失败本质所在层能否推理时引导治
1新旧物体混合时，模型偏向抓见过的旧物体，忽略指令要的新物体Familiarity Bias 走捷径，旧物体特征激活强、抢占动作感知→动作的注意力/选择层✓ 有戏

4夹爪开合不匹配（小→大物体）数据覆盖不足（二值化可绕过）动作细节—

三、推荐课题

指令感知的注意力引导（Instruction-Aware Attention Steering for Frozen VLA）

解决的 OOD：第 1 类（新旧物体混合偏向旧）+ 第 3 类（方位指令跟随不了）——真机高频、全竞品空白

理由（为什么选这个）

① 真空白
所有竞品在动作/目标/像素层；无人在 VLA 内部注意力做推理时 steering（NLP 有 activation steering，机器人空白）
② 真机高频
"新旧混合偏向旧"是部署 VLA 的必然场景（真机一手观察），不是实验室构造的 toy problem
③ 不 overclaim
诚实说解决选择层 OOD（第1/3类），不碰行为先验 OOD（第2类，那是引导盲区+大机构主场）
④ 可视化强
主图=引导前后 attention map 对比（从关注旧物体→关注新物体），一张图讲清故事

初步方案

观测 I_t + 指令 l → VLM 解析"指令要抓的目标物体是哪个/放到哪个方位" → 在冻结 π0.5 视觉编码器的 attention 层： 对"目标物体对应的 token"加权提升（或抑制旧物体 token） → VLA 的动作生成"听到"了正确物体的信号 → 动作自然从"抓旧物体"变成"抓指令要的新物体" 不改 π0.5 权重、不重训、不加动作空间梯度 本质 = 推理时 attention re-weighting，从 NLP activation steering 迁移到机器人

分阶段路线（由简到复杂）

P0 ★ 探针实验：可视化 attention（生死命门）
P1 VLM 定位目标 → attention 提升目标 token
P2 方位指令 → 空间位置 attention 强化
P3 IBVS 思想：object 误差→注意力修正
P4 松灵双臂真机验证

期望效果

维度预期

对比竞品在 ARRO mask 不了、VLS/OmniGuide 几何引导不解决的"物体选择 OOD"上胜出

四、我们的方案与所有竞品的区别

竞品操作层我们的差异
DynaGuide / VLS / OmniGuide动作空间梯度我们在 VLA 内部注意力层，源头纠正"听哪个物体"
ARRO像素层 maskARRO mask 任务无关像素；我们 steer 任务核心物体间的注意力竞争
DWYS/SEAL动作重采样+验证SEAL 依赖 forward rollout oracle；我们不需仿真，直接 steer 注意力
Goal-VLA / VERA替代 VLA我们保留冻结 VLA 学习先验，轻量引导不替代
SuSIE / GHIL-Glue目标层我们不生成子目标，直接 steer VLA 内部（不需训低层策略）
NLP Activation SteeringLLM 内部首次从 NLP 迁移到 VLA/机器人场景，解决物理 OOD 而非 safety
一句话核心 claim：VLA 在新旧物体混合部署场景的失败，不是因为"不认得新物体"，而是训练高频物体的注意力优势（familiarity bias）劫持了动作生成。我们首次在推理时 steering VLA 的内部视觉注意力，把"听哪个物体"的选择权从训练频率还给指令语义。

五、IBVS 思想的保留（方向 B 查证后的转化）

背景：用户有 IBVS 经验，提出"学一个通用的图像误差→运动映射攻行为 OOD"。查证发现 VERA(MIT 14B) 已做了"视频 rollout + Jacobian IDM 反解动作"的合体，正面撞车。但 IBVS 思想可以轻量用在注意力层——
传统 IBVS：δ运动 = J⁺ · (特征_目标 - 特征_当前) → 特征误差驱动运动 用户变体：δ_attention = f(表示_目标物体 - 表示_VLA当前关注) → 表示误差驱动注意力修正 差异：VERA 用 Jacobian 在动作空间反解；我们用误差信号在注意力空间做修正 保留 IBVS 的"误差驱动"核心思想，但作用层从运动→注意力

这是 P3 阶段的探索内容（非必须），主方案 P0-P2 不依赖它。
